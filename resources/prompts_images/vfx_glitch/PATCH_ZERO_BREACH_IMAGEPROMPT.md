# Patch-Zero Breach (VFX) — image prompt (nano banana)

A stylized 3D visual-effect study: a "reality breach" glitch. A jagged tear ripped into space, through which the world fractures into corrupted digital artifacts — broken shattered polygons, scan-line distortion, flickering magenta and cyan error-glitch bands, floating fragments of broken code and static, datamosh smears. It looks wrong, unstable and corrupting, like a system error tearing through reality.

Art style: cel-shaded / stylized 3D VFX, cyber-gothic, high contrast. Render the effect centered on a plain dark neutral (near-black) background so the glowing magenta/cyan glitch reads clearly, with soft even ambient light. This is an effect/VFX reference, not a solid object.
